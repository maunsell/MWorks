type(value, type)

ScarabConnection

The server(), server_root(), and serve() methods in an ScarabConnection class are the "real" methods for any particular protocol. Scarab figures out which protocol is going to be used, creates the appropriate ScarabConnection instance, and then forwards those methods to the new instance. Users don't call those directly.

server(url [, options])
Creates a new connection using the transport given in url. options is a dictionary containing options specific to each implementation.

server_root(scarab_url [, options ])
Creates a new connection using the transport given in url and return a proxy for the server's root object. options is a dictionary.

serve(scarab_url [, options ])
Creates a server endpoint for url. options is a dictionary. The server is not started until start() or accept() is called.
